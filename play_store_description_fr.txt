DESCRIPTION COMPLÈTE (Français - fr-FR)
=====================================

⚠️ APPLICATION NON OFFICIELLE - Cette application n'est pas affiliée à la SNCF ni à aucune entité gouvernementale. Il s'agit d'un projet indépendant open-source utilisant des données publiques.

Consultez vos horaires de trains TER en temps réel, directement sur le quai.

Prochains départs TER affichés avec retards et informations de voie (si disponibles). Simple, rapide, sans compte ni publicité.

FONCTIONNALITÉS
- Horaires temps réel (départs, retards, suppressions)
- Suivi multi-trajets (domicile-travail, etc.)
- Widget écran d'accueil
- Mode hors-ligne avec cache
- Tri automatique matin/soir
- Mode sombre

VIE PRIVÉE
- Aucun compte requis
- Aucune publicité
- Aucun traqueur
- Toutes vos données restent sur votre appareil

DONNÉES & SOURCES OFFICIELLES
- API utilisée : Navitia (SNCF) - https://api.sncf.com
- Documentation API : https://doc.navitia.io
- Portail développeur : https://numerique.sncf.com
- Open Data SNCF : https://www.sncf.com/fr/groupe/open-data
- Portail national : https://transport.data.gouv.fr
- Accès aux données via proxy anonyme ou clé API personnelle (utilisateurs avancés)
