# Store Assets

Assets pour les app stores (Play Store, App Store) - **NON EMBARQUÉS dans l'app**.

## Fichiers

- `icon-512.png` - Icône haute résolution (512x512) pour Google Play Store
- `feature-graphic.png` - (Futur) Bannière Google Play Store (1024x500)
- `screenshots/` - (Futur) Screenshots spécifiques aux stores

## Usage

Ces fichiers sont utilisés uniquement lors de la soumission aux stores.
Ils ne sont PAS inclus dans le build de l'application (APK/IPA).
